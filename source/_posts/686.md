MUI 团队近日正式发布了 **Base UI 1.0**。作为一个“无样式（Unstyled）”的 React 组件库，它旨在为开发者提供最纯粹的逻辑和可访问性支持，而将外观设计的完全控制权交还给开发者。

### 🎨 真正的定制化自由

Base UI 1.0 包含了 35 个经过精心打磨的无障碍组件：
*   **零耦合样式**：不强制使用任何 CSS 框架，无论是 Tailwind、CSS Modules 还是 Emotion 都能完美兼容。
*   **无障碍保障**：所有组件均符合 WAI-ARIA 标准，开箱即用。

### 🚀 长期维护承诺

作为 MUI 生态的重要组成部分，官方承诺将对 Base UI 进行长期的维护和性能优化。对于追求独特视觉体验且对代码体积有严格要求的企业级项目来说，Base UI 1.0 提供了一个极具吸引力的平衡点。
