Anthropic 近日正式发布了其最强模型 Claude Opus 4.6。作为该系列的最新迭代，Opus 4.6 在深度任务处理、智能体编码以及直接操作计算机（Computer Use）方面展现出了统治级实力。

### 🚀 跨越式的性能提升

与前代相比，Opus 4.6 在多个核心领域实现了大幅领先：
*   **编码之王**：在 HumanEval 等权威编码测试中，Opus 4.6 的评分远超同类竞争模型。
*   **计算机操作**：优化了对屏幕视觉的理解和对键鼠指令的精准执行，能够更流畅地完成跨应用办公任务。
*   **搜索与金融**：在处理长文档、复杂金融报表和实时网络搜索整合方面，其推理准确度提升了约 30%。

### 🧠 专注深度任务

Anthropic 表示，Opus 4.6 的设计初衷是成为真正能“干活”的智能体。它不仅能回答问题，还能在复杂的软件环境中进行自主调试和流程管理。

“Opus 4.6 证明了我们离真正的通用智能体更近了一步，”Anthropic 团队在官宣中提到。目前，该模型已通过 API 和 Claude Pro 订阅频道上线。
