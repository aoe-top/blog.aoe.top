知名 Python 库 Pydantic 的开发团队近日开源了新项目 **Monty**。这是一个使用 Rust 编写、专为 AI 代理安全运行代码而设计的极简 Python 解释器。

### 🚀 安全与速度并重

Monty 的设计初衷是解决 AI 在执行生成代码时的安全隐患：
*   **严格沙箱**：基于 Rust 的内存安全性，Monty 能够限制 AI 代码访问系统敏感资源。
*   **轻量化**：剔除了传统 Python 中与 AI 任务无关的冗余部分，启动速度极快。

对于构建“代码解释器”功能的开发者来说，Monty 提供了一个既能保持 Python 语法亲和力，又具备工业级安全性的运行底座。
