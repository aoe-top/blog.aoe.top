全球职场社交巨头 LinkedIn 近日宣布，已成功重构其静态应用安全测试（SAST）流水线。新系统深度集成了 GitHub Actions、CodeQL 以及 Semgrep，实现了数千个代码库的标准化扫描。

### 🛡️ “左移”战略的实践

此次升级的核心目标是进一步推进“安全左移”：
*   **自动化响应**：每次代码提交都会触发定制化的 GitHub Actions 流动，确保漏洞在进入生产环境前被拦截。
*   **可观测性提升**：通过自定义工作流，LinkedIn 的安全团队能够实时监控全公司范围内的安全态势。

### 🛠️ 工具链的整合

LinkedIn 不再依赖单一的闭源方案，而是转向了更具弹性的开源生态：
*   **CodeQL** 用于深度语义分析。
*   **Semgrep** 用于快速匹配已知的模式。

这种混合模式在保证扫描深度的同时，也极大缓解了开发者在 CI/CD 过程中的等待焦虑。
