## 安装

### 查看是否安装了 ssh-server 服务
```
dpkg -l | grep ssh
```

### 安装ssh-server服务

```sh
sudo -i
apt-get install openssh-server
```

### 再次查看安装的服务
```
dpkg -l | grep ssh
```

### 确认ssh-server是否启动

```
ps -e | grep ssh
```

## 登陆SSH（Linux）

### Windows登录

直接运行cmd ，然后在里面输入
```
ssh username@192.168.1.103
```
> 其中，username为192.168.1.103机器上的用户，需要输入密码。  
> 断开连接：exit
